# SIP AI Playground

Project pembelajaran AI selama PKL di Kemendag.

## Tujuan

Project ini dibuat untuk mempelajari bagaimana membangun sistem AI dari awal hingga akhir.

Target project:

- Memahami SIP
- Belajar Backend (FastAPI)
- Belajar PostgreSQL
- Belajar Docker
- Belajar Git & GitHub
- Belajar RAG
- Belajar LLM
- Membuat Chatbot Regulasi
